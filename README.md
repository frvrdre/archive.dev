# archive.dev

A modern personal website built with **Next.js** and **Tailwind CSS**.

🔗 Live site: https://archive-dev-chi.vercel.app/

---

## About

**archive.dev** is my personal space on the web — a place to build, experiment, and improve as a front-end developer.

The project focuses on clean UI, smooth interactions, and a minimal dark aesthetic while giving me space to showcase projects, ideas, and ongoing development work.

---

## Built With

- **Next.js**
- **TypeScript**
- **Tailwind CSS**
- **Vercel**

---

## Features

✨ Clean responsive layout  
🎵 Built-in music player  
🌑 Minimal dark UI  
⚡ Fast performance with Next.js  
🧩 Reusable component structure  
📱 Mobile-friendly design  

---

## Project Structure

```txt
app/          → pages & routing
components/   → reusable UI
context/      → music player state
public/       → static assets
```

---

## Running Locally

Clone the project:

```bash
git clone https://github.com/frvrdre/archive.dev.git
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Then open:

```txt
http://localhost:3000
```
